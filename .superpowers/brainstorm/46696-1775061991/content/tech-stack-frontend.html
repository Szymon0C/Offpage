Sekcja 5: Frontend tech stack

Frameworki i biblioteki po stronie webview

Frontend Stack
⚛️
React 19
UI framework — największy ekosystem, łatwy do rekrutacji, bogata biblioteka komponentów
🔷
TypeScript
Type safety — szczególnie ważne przy IPC z Rust i postMessage z iframe
⚡
Vite
Bundler — Tauri oficjalnie rekomenduje, szybki HMR, zero config
🎨
Tailwind CSS 4
Stylowanie UI apki — szybki development, spójny design system. Nie wpływa na iframe (izolacja).
🐻
Zustand
State management — lekki, prosty, wystarczający. Projekty, chat, UI state.
🛣️
React Router 7
Routing — home/projekty/szablony/ustawienia. Prosty, sprawdzony.
Świadomie pominięte:
❌ Next.js — server framework, niepotrzebny w Tauri
❌ Redux — overkill na ten scope
❌ CSS Modules — Tailwind pokrywa potrzeby

Podsumowanie

  • React 19 + TypeScript + Vite — standardowy, dobrze wspierany stack dla Tauri
  • Tailwind CSS 4 — stylowanie UI apki (nie strony użytkownika — ta jest w iframe)
  • Zustand — lekki state management zamiast ciężkiego Reduxa
  • React Router 7 — nawigacja między widokami apki
